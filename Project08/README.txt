This is a Java program that will turn the .vm files into .asm files. The 
file is in the /src folder "VMTransltor.java".

(0) Compilation
If you haven't set up Java environment, please refer to Oracle's website, 

https://docs.oracle.com/javase/tutorial/essential/environment/paths.html

After the environment is ready, open the command line and simply type in 

"javac VMTranslator.java"

Then, you will see a "VMTranslator.class" executable file.

(1) Translation

To translate the vm files, type in

"java VMTranslator 'your-source-file.vm'"

You will see a 'your-source-file.asm' afterwards.

If you input a directory, the output .asm file will be under the directory
you inputed.

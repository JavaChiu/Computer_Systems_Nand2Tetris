This is my implementation of Nand2Tetris project11 JackCompiler.

0) Usage
Under the /src directory, there is a project11.jar executable jar file,
under command line type in 

java -jar project11.jar "target jack file/directory"

Then it will generate the out put vm files in the same directory.

1) Source and Packaging
The source code is under the /src/project10/ directory. With packaging, 
you will need to use an MANIFEST.MF or other jar cmd to package it. More
information can be found on Oracle website,

https://docs.oracle.com/javase/tutorial/deployment/jar/

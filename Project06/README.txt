This is a Java program that will turn the .asm files into .heck files. The 
file is in the /src folder "Assembler.java".

(0) Compilation
If you haven't set up Java environment, please refer
to Oracle's website, 

https://docs.oracle.com/javase/tutorial/essential/environment/paths.html

After the environment is ready, open the command line and simply type in 

"javac Assembler.java"

Then, you will see a "Assembler.class" executable file.

(1) Translation

To translate the asm file, type in

"java Assembler 'your-source-file.asm'"

You will see a 'your-source-file.hack' afterwards.

(2) If you encounter any problem, welcome to email at chungan@uchicago.com


ENJOY!
